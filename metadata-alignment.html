METADATA ALIGNMENT

METADATA ALIGNMENT

The goal of the metadata alignment is to show the relationships between the items metadata and the respective standards. Align means also compare. Compare the ways in which the stadards are able to describe a same entity through different perspectives, the ones attributed to the standards themselves. The metadata analysis has been divided into two parts: the entity alignement and the item alignement. The first one is a table where four entities (People, Date, Place and Subject) have been taken into account in order to be described by all the standards. The choice of the properties was based on the characteristics of the standards and on the items ones. Only a few properties could be described by all the standards and were representative of the metadata of our items. These properties are the ones shown in the first four tables.
In a second moment the items have been described through the ontologies or standards which could best describe them. In this case only three or four standards have been selected. Those ones are available in the last tables, named by type (audio, biography etc.).
